To start this project use: 
	- (node server.js) - static 
	- or (nodemon server.js)     <-------

Eg.
PS C:\Users\48725\verso> nodemon server.js
[nodemon] 2.0.4
[nodemon] to restart at any time, enter `rs`
[nodemon] watching path(s): *.*
[nodemon] watching extensions: js,mjs,json
[nodemon] starting `node server.js`
SERVER IS STARTING ON PORT,  8080


In your browser type localhost:8080 and you will get to the main webpage



HTML/CSS/JS files are in public folder. It's client side directory.
Server.js is our API to connect backend and server with frontend.
Mail.js is mail template.
